# 🌲 森林生态模拟

一款基于 HTML5 Canvas 的 2D 森林生态模拟游戏，核心实现**生物数量动态平衡**与**自然狩猎行为**。单 HTML 文件，零外部依赖。

---

## 🎮 快速开始

直接双击 `forest-ecosystem.html` 在浏览器打开，点击 **「开始模拟」** 即可运行。

> 支持 Chrome / Firefox / Edge / Safari 等现代浏览器。

## 🧠 老虎 AI — 4 态状态机

```
                    ┌──────────────┐
                    │   IDLE       │ 闲逛/搜索
                    │  · 不饿→溜达 │
                    │  · 饿→朝猎物 │
                    └──────┬───────┘
                           │ 发现猎物
                           ▼
                    ┌──────────────┐
                    │   STALK      │ 潜行 2.0
                    │  · 悄悄接近  │
                    │  · 猎物不察觉 │
                    └──────┬───────┘
                           │ 距猎物<130px
                           ▼
                    ┌──────────────┐
                    │   SPRINT     │ 冲刺 6.8
                    │  · 猎物逃跑  │
                    │  · 距<55px→   │
                    │    爆发 8.5  │
                    └──────┬───────┘
                    耐力耗尽│ 追上→进食
                           ▼
                    ┌──────────────┐
                    │  COOLDOWN    │ 喘息
                    │  · 50 tick后 │
                    │    恢复      │
                    └──────────────┘
```

**智能寻路**：没猎物时主动搜索地图中心区域，并记忆上次猎物位置。

## 🦌 猎物 AI — 3 层优先级

每 tick 检查（从上到下）：

| 优先级 | 行为 | 说明 |
|--------|------|------|
| **1** | 🐯 感知老虎 → 智能逃跑 | `smartFlee()`：沿墙滑行 + 角落恐慌冲刺 + 指数级边界排斥 |
| **2** | 📢 预警传递 | 附近同伴快速移动时跟随 |
| **3** | 🌿 觅食 | 找最近植物（HP>80%时不饿不吃）→ 没有则朝地图中心搜索 |

**智能逃跑算法**：
1. 主方向：远离老虎
2. 墙壁检测：如逃跑方向会撞墙，改为**沿墙滑行**
3. 角落恐慌：离两面墙都很近（<25px+<40px）→ **直接朝中心冲刺 ×1.3**
4. 边界指数排斥：越靠近边缘，往中心拉的力越大
5. 随机散开：±20° 防止扎堆

## 🔄 生态循环（每 tick 300ms）

```
1. 年龄 + 天计时
2. 记录种群历史（每5 tick，用于曲线图）
3. 老虎移动 → 4态状态机
4. 鹿/羊移动 → 3层优先级（逃跑→预警→觅食）
5. 捕食判定（距离<25px）
6. 食草动物吃植物（HP<80%才吃）
7. 饥饿系统（老虎饥饿累积，鹿羊自然消耗）
8. 衰老判定（超过寿命上限→老死）
9. 繁殖判定（各物种独立参数+密度制约）
10. 清理死亡个体
11. 植物再生
```

## 📁 代码架构

```
forest-ecosystem.html
├── 📄 HTML — Canvas + 侧栏面板 + 控件
├── 🎨 CSS — 暗色森林主题 + 响应式侧栏
└── 🧩 JavaScript
    ├── CONFIG         — 所有可调参数常量
    ├── Utils          — 工具函数
    ├── SoundManager   — Web Audio API 音效合成
    ├── Creature       — 生物基类（含动画属性）
    ├── Tiger          — 4态狩猎状态机 + 智能搜索
    ├── Deer           — 群居+警觉+智能逃跑
    ├── Sheep          — 群居+耐饿+智能逃跑
    ├── Plant          — 再生机制
    ├── Ecosystem      — 核心循环（更新/捕食/繁殖/饥饿）
    ├── Renderer       — Canvas 渲染（背景缓存+动物动画+曲线图）
    └── Main           — 事件绑定+滑条+预设
```
